# Pounce

A tiny, dependency-free GitHub Pages website featuring Marmalade, a cat who follows your cursor and pounces when you click.

## Features

- Smooth cursor following and edge-aware direction changes
- Eye tracking, breathing, and tail animations
- Click or tap pounce interaction
- Touch-screen and reduced-motion support
- Plain HTML, CSS, and JavaScript with no build step

Visit the live site through this repository's GitHub Pages link.
